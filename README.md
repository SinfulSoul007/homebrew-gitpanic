# Homebrew Git Panic

Homebrew tap for [Git Panic CLI](https://github.com/SinfulSoul007/gitpanic-cli).

## Installation

```bash
brew tap SinfulSoul007/gitpanic
brew install gitpanic
```

## Usage

```bash
gitpanic              # Open interactive menu
gitpanic undo         # Undo last commit
gitpanic fix-message  # Fix commit message
gitpanic --help       # Show all commands
```

## Update

```bash
brew update
brew upgrade gitpanic
```

## Uninstall

```bash
brew uninstall gitpanic
brew untap SinfulSoul007/gitpanic
```
